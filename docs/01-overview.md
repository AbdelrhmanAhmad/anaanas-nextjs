# 01 — نظرة عامة

## الهدف

إعادة بناء تجربة الصفحة الرئيسية على الموبايل لتعتمد على لوحة "اكتشاف"
(Discovery) مدمجة، مع رأس موبايل مخصّص، شبكة أقسام مضغوطة، لوحة ذكاء
اصطناعي، وشبكة عروض ترويجية — كلها متوافقة مع ثيم العلامة (Ananas).

## ثيم العلامة

تم توحيد الألوان عبر كل المكوّنات الجديدة لمحاكاة لوحة الـ
`select-country` وشاشة التحميل (cream + gold + coral + violet).

| المتغيّر       | القيمة      | الاستخدام                         |
| -------------- | ----------- | --------------------------------- |
| `--an-cream`   | `#fffdf7`   | خلفية عامة                        |
| `--an-cream-mid` | `#fff8eb` | تدرّجات داخل البطاقات              |
| `--an-peach`   | `#fff3dc`   | لمسة دافئة في الخلفيات             |
| `--an-gold`    | `#fecb01`   | لون العلامة الأساسي               |
| `--an-gold-soft` | `#ffe566` | تظليل سرّ ذهبي                    |

| `--an-orange`  | `#e76f51`   | تظليل ثانوي                       |
| `--an-violet`  | `#ba7bff`   | بطاقات الذكاء                     |

> ثيم عالمية حالياً، لذا التعديلات تظل معزولة.

## تايبوغرافيا (خط Cairo)

- الأحجام: غالبًا بين **9px و 13px** على الموبايل.
- الأوزان: **500 / 600** فقط؛ تجنّب 700/800 لأن Cairo يظهر ثقيلًا.
- المسافات السطرية: **1.2 → 1.35** لإحكام القراءة على الموبايل.
- تقطيع طويل: `text-overflow: ellipsis` على عنوان البطاقة + `-webkit-line-clamp`
  لتقييد عدد الأسطر في الوصف.

## مبادئ الحركة (Motion)

- **دخول القسم** عبر `motion/react` (Framer Motion) مع
  `useReducedMotion()`.
- **الحركة الدائمة** (idle / infinite) تبقى في **CSS** للأداء، وتُلغى
  جميعها داخل `@media (prefers-reduced-motion: reduce)`.
- الستاجر (`staggerChildren`) يُستخدم لظهور بطاقات الـ Dashboard تباعًا.
- تجنّبنا مكتبات إضافية (Bubble background / react-spring) للحفاظ على
  حجم الباندل.

## معايير عامة

- **شبكة 2×3 / 3×2 على الموبايل** بدل أعمدة 3 متساوية حين تتأثر القراءة.
- **بدون صور بكسلية إن أمكن** — استبدلنا الماسكوت بـ SVG inline.
- **إتاحة الوصول**: عناوين دلالية (`h2/h3`)، `aria-label` على الأقسام،
  `tabIndex={0}` على البطاقات، `focus-visible` بلون ذهبي العلامة.
